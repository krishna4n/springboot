Spring Boot

    STEP 1 : Choosen to use springboot 3.0.6 version for this project.

    STEP 2: Using https://start.spring.io, generated default frame work with version 3.0.6 and java 17 version and Added dependencies are Spring Web ,MySQL Driver, Spring Data JPA.

    STEP 3: Below are the application properties, 

    			server.port=8009
			//this will connect database
			spring.datasource.url=jdbc:mysql://localhost:3306/gk?useSSL=false
			spring.datasource.username=root
			spring.datasource.password=gk
			spring.jpa.database-platform=org.hibernate.dialect.MySQLDialect
			//spring.jpa.hibernate.ddl-auto=update
			spring.jpa.show-sql=true
			//This flyway will log all the database activities like github repository
			spring.flyway.baseline-on-migrate = true
			spring.jpa.generate-ddl-auto=false
